# วัตถุประสงค์  
เกมนี้ถูกสร้างขึ้นเพื่อทำให้การเรียนระดับมัธยมสนุกขึ้น
ฝึกความรับผิดชอบและการบริหารเวลา
พร้อมทั้งสร้างแรงจูงใจให้ผู้เล่นพัฒนาตัวเองผ่านระบบเกม  
# เกมภารกิจเสริมทักษะ
1.หน้า Home (หน้าแรก)
 • ชื่อเกม
 • คำอธิบายสั้น ๆ ของเกม
 • ปุ่ม เริ่มเกม
 • ปุ่ม ดูภารกิจ
 • แถบสถานะผู้เล่น (Level / EXP)

  ทำหน้าที่
 • เป็นจุดเริ่มต้นของเกม
 • สร้างความเข้าใจว่าเกมเกี่ยวกับอะไร


2.หน้า Profile (โปรไฟล์ผู้เล่น)

มีอะไรบ้าง
 • Avatar (รูปตัวละคร 2D)
 • ชื่อผู้เล่น
 • Level
 • EXP
 • สถิติการส่งงานตรงเวลา

ทำหน้าที่
 • แสดงความคืบหน้าของผู้เล่น
 • สร้างแรงจูงใจให้พัฒนา Level

⸻

3.หน้า Mission List (รายการภารกิจ)
 • รายการภารกิจแบบ Card
 • ชื่อภารกิจ
 • วิชา
 • เวลาเหลือ (Countdown)
 • สีสถานะ (เขียว / เหลือง / แดง)
 • ปุ่มเข้าแต่ละภารกิจ

ทำหน้าที่
 • ให้ผู้เล่นเห็นภาพรวมของงานทั้งหมด
 • ช่วยวางแผนจัดการเวลา

⸻

4.หน้า Mission Detail (รายละเอียดภารกิจ)
 • ชื่อภารกิจ
 • คำอธิบาย
 • เวลาเหลือแบบ Countdown
 • ปุ่ม เริ่มทำภารกิจ
 • เงื่อนไขรางวัล / โทษ

ทำหน้าที่
 • อธิบายภารกิจให้ชัด
 • ย้ำความสำคัญของเดดไลน์

⸻

5.หน้า Quiz / Mini Game
 • คำถามทีละข้อ
 • ตัวเลือกคำตอบ
 • ตัวจับเวลา (ถ้ามี)
 • ปุ่มยืนยันคำตอบ

ทำหน้าที่  
ประเมินความรู้ของผู้เล่น  
ทำให้การเรียนรู้เป็นเกม


⸻

6.หน้า Result / Submit

มีอะไรบ้าง
คะแนนที่ได้
สถานะการส่ง (ตรงเวลา / สาย)
EXP ที่ได้รับ
ปุ่มกลับหน้า Mission List

ทำหน้าที่  
แสดงผลลัพธ์ทันที  
ให้ผู้เล่นเห็นผลจากการจัดการเวลา
